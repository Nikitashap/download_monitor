# Скрипт для отслеживания скорости загрузки игр в Steam

Данный скрипт отслеживает с какой скоростью загружается игра в Steam. Этот скрипт работает только на MacOS. Информация о скорости загрузки выводится раз в минуту, на протяжении пяти минут. Если загрузка была остновалена или окончена, программа выведет эту информацию. Скрипт берет информацию из log файлов Steam и работает с ней при помощи регулярных выражений.
